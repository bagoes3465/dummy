🔧 MODEL ARCHITECTURE

  Input Image (224×224)
         ↓
  ResNet50 Backbone (ImageNet Pretrained)
         ↓
  Feature Extraction (2048 features)
         ↓
  Classification Head:
    • Dense(2048 → 512) + BatchNorm + ReLU + Dropout
    • Dense(512 → 256) + BatchNorm + ReLU + Dropout
    • Dense(256 → 7 emotions)
         ↓
  Output: 7-class Logits → Softmax → Probabilities
